## git add

```
git add .
git add *
git add --all
```
Добавить в индекс все изменённые файлы. Все приведённые варианты команды равнозначны.

```
git add file_name.txt
```
Добавить в индекс указанный файл.

```
git add folder_name/*.*
```
Добавить в индекс все файлы из указанной папки.
