## git branch

```
git branch -a
```
Показать имеющиеся ветки.

```
git branch <ветка>
```
Создать новую ветку не переходя в неё.

```
git branch -b <ветка>
```
Создать новую ветку и перейти в неё.

```
git branch -d <ветка>
```
Удалить ветку.
